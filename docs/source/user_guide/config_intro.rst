Config Introduction
===================

SATGL provides customizable parameters for directing the experiment workflow including data handling, partitioning, model
training, and analysis. Users can fine-tune these configurations based on their specific needs and priorities.

How to set config?
-----------------------------

SATGL supports three types of parameter configurations: Config files, Parameter Dicts and Command Line.
The parameters are assigned via the Configuration module.
